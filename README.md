# Keylogger 

Proyecto hecho  con fines educativos
## Requisitos
- Python 3.10+
- Permisos de Administrador 
- Descargar la libreria keyboard: https://github.com/boppreh/keyboardEnlaces

## Instalación
pip install -r requirements.txt`

## Ejecución
python main.py`
Para detener el programa, presiona la tecla "ESC".
